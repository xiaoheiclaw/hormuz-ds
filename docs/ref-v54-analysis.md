## §0 核心公式

这个冲突的持续时间和油价路径，最终由三个物理问题决定。

```
总缺口 = ∫₀ᵀ [ 毛缺口 − Buffer(t) ] dt    [mbd·天 = 累积库存损失]
```

拆解：

| 公式组件 | 含义 | 由谁决定 |
|---|---|---|
| **T = T₁ + T₂** | 中断持续时长 | Q1（主动威胁消除）+ Q2（扫雷清除），串联关系 |
| **毛缺口 ≈ 16 mbd** | 20.1 mbd × ~80% 有效中断率 | Q1 + Q2 联合锁定中断率（阶跃函数） |
| **Buffer(t)** | 缓冲爬坡函数 | Q3 独立运行，分段：D1–14 ~1.5 mbd → D14+ ~7 mbd |
| **净缺口(t) = 16 − Buffer(t)** | 每日实际供应缺口 | 分段变化，非恒定 |

因果链：**(主动威胁 + 被动威胁) → 有效中断率(阶跃) → 毛缺口 16 mbd**；扣除爬坡缓冲 = 净缺口路径 → 驱动油价。

### 有效中断率：历史校准

设定冲突僵持期中断率 ~80% 并非主观臆测：

- **1984年袭船战**激化初期，霍尔木兹流量24小时内暴跌约70%
- **2026年3月初**实际通行量跌至仅剩13艘（约正常运量的8%，即92%中断率）

稳态取80%保守估计。中断率在T期间近似恒定，T结束时阶跃回落至 <10%（非渐变）——只有物理清扫完成（Q2验证），保险才恢复承保。

### 变量分类法（状态空间模型）

系统内所有信息按性质分为六类，每类有不同的更新规则：

| 变量类别 | 定义 | 更新规则 | 举例 |
|---|---|---|---|
| 常数 | 所有情景、所有时间不变 | 不更新 | 海峡航道宽度、需清扫面积 |
| 参数 | 战争时间尺度内锁定 | 重大情报修正或制度变更时才改 | 水雷初始库存、SPR泵送最快13天 |
| 状态变量 | 随 stock-flow 动态演化 | 每日/每周 | 水中水雷数、ASCM剩余存量 |
| 观测 | 状态变量的带噪声测量值 | 每日采集 | 袭击频率、AP保费率、GPS欺骗集群 |
| 控制/决策输入 | 各方 actor 的决策 | 信号驱动 | 护航命令、调停框架、SPR释放令 |
| 校准参照 | 用于校准严重度或时间线 | 不更新（固定历史事实） | 1984袭船战、1991海湾战争、1950元山港 |

## §1 中断时长 T = T₁ + T₂

T 由两个物理问题串联决定。Q1 达到阈值后 Q2 才启动——扫雷舰在主动威胁活跃区域无法安全作业。

### Q1 → T₁：主动威胁衰减速率

IRGC 的导弹、快艇、无人机在美军持续压制下以什么速率消耗。决定"什么时候扫雷可以开始"和"什么时候护航可行"。

**关键阈值**：袭击频率 < 2次/天 = Q2（扫雷）的启动条件。这是 Q1 对 Q2 的因果传导点。

| 常数 | 值 | 含义 |
|---|---|---|
| 伊朗北岸地形 | 沿岸山地提供天然隐蔽；海峡北岸全是伊朗领土 | 机动发射平台可藏匿，压制需不间断空中巡逻 |
| 海峡航道宽度 | 可通航航道约9km（两条3km航道 + 3km缓冲） | 狭窄航道使主动威胁杀伤效率更高 |

## §2 净缺口(t) = 16 mbd − Buffer(t)

T 期间毛缺口被 80% 中断率锁定在 ~16 mbd（近似恒定）。净缺口取决于 Q3 缓冲以什么速度到位。

### Q3：缓冲机制与爬坡函数

| 缓冲机制 | 硬约束 | 受约束最大值 | 启动时间 |
|---|---|---|---|
| 沙特东西管道 | 延布港装卸瓶颈与国内炼厂截留 | 增量 1.9–3.0 mbd | 3–5天切换 |

**爬坡函数(t) 分段**：

- **D1–D14**（极速爬坡期）：缓冲 ~1.5 mbd → **净缺口 ~14.5 mbd**
- **D14–D30+**（爬坡放缓趋稳态）：缓冲 ~7 mbd → **净缺口 ~9 mbd**

## §4 博弈层：Schelling Signal Sheet

> **层级定位**：博弈层只影响 T（改变路径A/B/C的概率权重），不直接驱动 Buffer。Buffer(t) 是独立的物理过程（Q3驱动）。路径C中 Buffer 崩溃（如富查伊拉遇袭）是物理事件，与博弈失控相关但因果独立——失控提高两者同时发生的条件概率，并非直接因果链。物理时间常数仍设定T的硬下界，政治决策无法突破。

### Schelling 三诊断

**① 承诺锁定松动？**

承诺机制（Commitment Device）通过削弱自身回撤能力来增强威胁可信度。

- **IRGC侧**：去中心化指挥结构构成典型承诺机制。即便莫杰塔巴本人有意降级，也未必能将意图稳定传导至整个执行体系——组织结构已大幅压缩"停下来"的操作空间。
- **美方侧**：Trump"unconditional surrender"表述具有类似性质。公开宣称无妥协空间，缩小了未来回旋余地，后续让步面临极高政治成本。
- **持久性约束**：承诺机制持久性取决于物理支撑。IRGC声称六个月弹药，但有战略夸大的强烈动机。首轮打击已摧毁部分设施，实际持续封锁能力可能在2–8周后显著衰减——承诺机制失去物理支撑就会自动解锁。
- **诊断信号**：任何一方信号包出现内部不一致（美方打击继续但优先级转向护航 / 伊方IRGC与政治层表态分裂）→ T缩短预期，路径A↑

**② 失控风险上升？**

边缘策略（Brinkmanship）——"留给机会的威胁"。

- **分散化放大"机会"**：IRGC执行链条的分散化放大了失控成分。地方指挥官可能自行扩大攻击目标，德黑兰未必能保证升级不发生。真正施压的不只是升级意图，而是"系统内部某些行为体可能做出连中央都难以约束的行动"。
- **分散化的双重性**：同样的结构也意味着各省级单位可能在弹药耗尽后各自寻求地方性停火，而非保持统一对抗。分散化既是承诺的强化，也是碎片化的前兆——两种演化当前都合理。
- **诊断信号**：未经决策但导致事实升级的行为（IRGC目标溢出航运 / 误击事件 / 指挥链失控信号）→ T延长预期，路径C↑↑

**③ 外部聚点注入？**

聚点（Focal Point）成立的前提是双方至少共享某种基本认知框架。当前面临三重障碍：双方无共享的"到此为止"定义；对手不统一（IRGC内部碎片化）；缺乏共享战略文化。内生聚点极难形成。

但聚点不一定需要内生产生。三条外部注入路径：

- **路径A：中国调停（20–30%）**——中国是霍尔木兹最大利益相关方（84%流量流向亚洲）。可能结构：中国担保的有限通行安排，IRGC允许中国船队护航通过，美国默许，逐步扩展。约束：中国可能不愿承担调停失败的声誉风险。
- **路径B：安理会框架（10–15%）**——决议设定时限（如72小时停火供人道通行）可创造外部协调点。约束：美国可能否决不含伊朗投降条件的决议。
- **路径C：有限通行换局部停火（25–35%）**——非正式安排：IRGC停止攻击中立国船只，换取美方暂停对伊朗本土特定目标打击。不需要任何一方承认失败，只需双方解读为"战术调整"。IRGC弹药紧张、美国中期选举压力上升后可行性上升。

**博弈结论**：当前处于"双重承诺锁定"——谢林框架中的高危构型。聚点需要外部注入且启动条件苛刻。退出路径存在，但需催化剂，执行过程可能反复不稳定。
